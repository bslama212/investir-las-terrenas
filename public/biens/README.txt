PHOTOS DES BIENS
================
Déposez les photos de l'Unité B ici :  public/biens/dream-house-b/
Nommage conseillé : 01.jpg, 02.jpg, 03.jpg ... (la 1re = photo principale / hero).

Puis renseignez-les dans la fiche :
  src/content/properties/dream-house-unite-b.md  ->  champ "images"

Exemple :
images:
  - { src: "/biens/dream-house-b/01.jpg", alt: "Terrasse de l'Unité B" }
  - { src: "/biens/dream-house-b/02.jpg", alt: "Salon / cuisine ouverte" }
  - { src: "/biens/dream-house-b/03.jpg", alt: "Salle de bain" }

La galerie et le visuel d'en-tête s'afficheront automatiquement.
